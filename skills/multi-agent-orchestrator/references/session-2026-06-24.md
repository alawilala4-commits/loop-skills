## Key Decisions

- **Commit-first pattern**: Coder committed code before Tester ran tests. Commit hash verified before handoff to Logger.
- **Tester isolation**: Tester only ran existing tests — did not modify code or add test cases.
- **Reviewer read-only**: Reviewer used `file` toolset only — no code modifications.
- **Sequential handoff**: One agent at a time, Orchestrator validated each result before proceeding.
- **Context minimization**: Each agent received only what it needed. Coder didn't see test results. Tester didn't see review criteria.
