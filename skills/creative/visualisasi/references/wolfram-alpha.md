# Wolfram Alpha Quick Reference

## URL Format
```
https://www.wolframalpha.com/input?i={encoded_query}
```

## Categories
- **Math:** `integrate`, `derivative`, `solve`, `plot`, `limit`, `series`
- **Data:** `population`, `GDP`, `weather`, `unemployment rate`
- **Conversion:** `USD to IDR`, `celsius to fahrenheit`, `km to miles`
- **Units:** `height of Eiffel Tower`, `speed of light`, `density of gold`
- **Date:** `next Friday`, `days until Christmas`

## Embedding Options
1. **iframe (recommended for web):** Direct embed, interactive
2. **API (advanced):** Needs Wolfram Alpha API key, returns structured data
3. **Screenshot API:** `https://api.wolframalpha.com/v1/simple?i=query&appid=YOUR_APPID`

## Common Queries (Indonesian Context)
```
population of Indonesia
GDP of Indonesia
weather in Jakarta
distance from Jakarta to Surabaya
Indonesia exports inflation rate
capital of West Java
```

## Error Handling
- Kalau hasil tidak muncul: query mungkin terlalu vague
- Ganti query lebih spesifik
- Kalau iframe diblokir: fallback ke screenshot API atau direct link
